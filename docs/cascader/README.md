# Cascader - 级联选择器
当一个数据集合有清晰的层级结构时，可通过级联选择器逐级查看并选择。

## 基础用法
默认 click 触发子菜单

<ClientOnly>
  <cascader-demo type="basic"/>
</ClientOnly>

## 默认值
默认值通过数组的方式指定。

<ClientOnly>
  <cascader-demo type="default"/>
</ClientOnly>

## 动态加载次级选项
当选中某一级时，动态加载该级下的选项。
<ClientOnly>
  <cascader-demo type="dynamic"/>
</ClientOnly>
